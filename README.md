# 이음 (EUM) — 문화예술교육 수요·공급 연결 지도

> 2026 한국문화예술교육진흥원(ARTE) 공공데이터 활용 아이디어 공모전 출품작

ARTE 개방 공공데이터로 **문화예술교육 공급의 사각지대(대상군 × 지역)를 지도에 드러내고**,
소외 지역·대상을 인근 공급주체와 **연계**하는 플랫폼.

## 핵심 발견 (ARTE 개방데이터 분석)
- 프로그램 1,311건을 지오코딩 분석 → 전국 250개 시군구 중 **60곳은 프로그램 0건**
- **장애인 49건·유아 27건**으로 특정 대상군이 구조적 사각지대
- 시도 간 격차 **경기 222 vs 세종 9 (24배)**

## 활용 데이터
- (필수) ARTE 문화예술교육 프로그램 목록 / 단체 목록 / 사업단위별 강사수 통계 (공공데이터포털)
- (선택) 카카오맵 Local API(지오코딩), 통계청 행정경계

## 기술 스택
- Web: Next.js 14 + Tailwind + 카카오맵 JS SDK
- Data: Python (pandas, shapely) ETL / 지오코딩 / 시군구 집계

## 구조
```
eum/
├─ web/          # Next.js 지도 앱 (Vercel 배포 대상)
│  └─ public/    # sigungu.geojson, programs.json (집계 결과)
├─ data/         # 수집·지오코딩·집계 스크립트
└─ assets/       # 분석 차트(제안서용)
```
